1. Запускаем terminal, переходим в папку с проектом и инициализируем локальный репозиторий с помощью команды `git init`.

2. Создаем новый удаленный репозиторий на платформе Git (например, на GitHub) с именем, соответствующим имени нашего проекта.
Создаем файл README.md

3. Связываем локальный и удаленный репозитории с помощью команды `git remote add origin <URL удаленного репозитория>

4. Проверяем связь между локальным и удаленным репозиториями с помощью команды `git remote -v`. В результате должны увидеть URL удаленного репозитория.

5.
	- Добавляем все измененные файлы в ваш локальный репозиторий с помощью команды `git add .`
        - Делаем коммит изменений с помощью команды `git commit -m "Описание ваших изменений"`
        - Отправляем коммиты в удаленный репозиторий с помощью команды `git push origin master`
6. Готово

7. Решаем Конфликт: 
        - git checkout main
        - git merge master
